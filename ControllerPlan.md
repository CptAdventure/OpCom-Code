Code for the controls:

- Hook: Servo
  - Lower Hook: LB
  - Raise Hook: LB

- Vertical Claw Movement: Motor
  - Lower Claw: D-Pad Up
  - Raise Claw: D-Pad Down

- Claw Grab: Servo
  - Open Claw: RB
  - Close Claw: RB

- Claw Extension: Motor
  - Extend Claw: D-Pad Left
  - Retract Claw: D-Pad Right

- Intake: Motor
  - Start Intake: A
  - Stop Intake: A
  - Unjam Intake: B
